# PIM (Projeto Integrador Multidisciplinar)

Registros do desenvolvimento do projeto semestral.

## Conteúdo
- Requisitos funcionais e não funcionais
- Diagramas (casos de uso, classes, etc.)
- Decisões de arquitetura
- Organização do projeto

## Objetivo
Documentar a evolução do projeto e as decisões tomadas durante o desenvolvimento.

## Formato dos registros
- Atualizações do progresso
- Definições importantes
- Problemas e soluções
