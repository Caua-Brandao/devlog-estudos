# Dev Log – Estudos em C#

## Conteúdo estudado

* StreamWriter (escrita de arquivos de texto)
* using block (garantia de fechamento de objetos IDisposable)
* Directory e Path (criação de pastas e manipulação de caminhos)
* Exercício de fixação: leitura de CSV e geração de arquivo de saída summary.csv

## Aprendizados

* `Split(',')` para separar campos de uma linha CSV
* `double.Parse()` e `int.Parse()` para converter string em número (cast não funciona)
* `CultureInfo.InvariantCulture` para evitar problemas com separador decimal
* `File.CreateText()` para criar arquivo novo do zero
* O `while (!sr.EndOfStream)` itera por linha, não por campo — cada `ReadLine()` traz uma linha inteira
* Estrutura de `using` aninhados: StreamWriter externo, StreamReader interno

* Entender que o array `fields` é recriado a cada iteração do loop
